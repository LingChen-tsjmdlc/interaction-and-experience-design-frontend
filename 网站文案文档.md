# 阿里云千问互动广告 - 网站文案文档

## 项目概述

**项目名称**：阿里云千问互动广告

**命题来源**：全国大学生广告艺术大赛（大广赛）

**官方网址**：sun-ada.net

**核心定位**：展示阿里云千问 AI 能力的互动宣传网页，通过沉浸式视觉体验和多语言支持，向用户介绍千问家族模型的强大功能。

## 技术特点

### 核心技术栈

| 技术            | 用途                   |
| --------------- | ---------------------- |
| Next.js 15      | React 框架，服务端渲染 |
| TypeScript      | 类型安全               |
| Tailwind CSS v4 | 样式系统               |
| HeroUI          | UI 组件库              |
| Framer Motion   | 动画库                 |
| react-i18next   | 国际化                 |
| Iconify         | 图标系统               |

### 视觉与交互

- **背景动画**：Aceternity UI BackgroundLines 动态网格背景
- **卡片效果**：CometCard 聚光灯跟随效果
- **文字动画**：BlurText、TextAnimate、SplitText 多种文字动效
- **交互动画**：AnimatedBeam 光束连接、AnimatedList 列表动画
- **深色/浅色模式**：next-themes 实现主题切换
- **响应式设计**：支持移动端、平板、桌面端自适应

### 多语言支持

- 支持语言：中文、英文、日文
- 实时切换：无需刷新页面
- 翻译覆盖：导航栏、Hero 区域、功能介绍、模型展示、弹窗等全部内容

### 动画设计理念

- **分阶段入场**：Hero 区域 7 个步骤（Logo → 标题 → 副标题 → 装饰线 → 输入框 → 快捷按钮 → 开始按钮）
- **交错动画**：功能介绍和模型展示使用 staggerChildren 实现流畅的序列动画
- **视差滚动**：useInView 监听元素进入视口，触发动画重播
- **交互反馈**：卡片 hover 效果、按钮点击反馈、输入框展开动画

## 文案创作原则

1. **简洁有力**：标题控制在 2-6 个字，描述控制在 1-2 句话
2. **技术准确**：模型名称和参数严格按照官方资料
3. **用户导向**：强调功能价值和用户收益，而非技术细节
4. **一致性**：中英文翻译保持语义对等，专业术语统一
5. **适配多语言**：避免使用文化特定隐喻，确保三语言版本都能自然理解
